Allows the use of a shortcode to embed Fluxey surveys into your site.

== Description ==

The Fluxey WordPress Shortcode Plugin gives you an easy way to embed your Fluxey surveys into your site. You can also find this code on the Fluxey Share page.

= Usage: =

`[fluxey id="A1B2C3D4"]`

= Extra Options: =

If you are not happy with the defaults you can set the height and width of the survey using the extra parameters below:

`[fluxey id="A1B2C3D4" width="500px" height="800px"]`

== Installation ==

To install manually: copy the folder "fluxey-wordpress-shortcode" into the /wp-content/plugins/ folder. Then go to the Plugins area of the Admin and activate.

Otherwise, search for "Fluxey Wordpress Shortcode Plugin" from the admin area of your WordPress site in Plugins > Add New.
